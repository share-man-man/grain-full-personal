# grain-full-personal

### 项目演示地址
[界面展示地址(国内)](http://shuxiaoman.gitee.io/grain-full-personal/).

## 安装依赖包：需要设置成淘宝的镜像才能下载
```
yarn config set registry https://registry.npm.taobao.org/
yarn install
```

### 运行项目
```
yarn dev
```

### 编译项目
```
yarn build
```

### 代码校验
```
yarn lint
```
